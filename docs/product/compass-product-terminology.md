# Compass product terminology

_Date: 2026-06-19_

This note separates current and future product language so the app can keep the existing live analytics experience without confusing it with future Compass Book work.

## Compass Book

Compass Book is the future deep, six-chapter curriculum for durable self-understanding and life direction. It is not implemented in the Quest Pulse rename PR.

## Quest Pulse

Quest Pulse is the existing live analytics experience formerly shown to users as Quest Compass. It synthesizes Life Wheel check-ins, goals, habits, current quest information, balance, momentum, trends, and attention signals.

## Old Island Run Compass

The old Island Run Compass is the legacy 11-phase Compass system used by Island Run. It is retained temporarily and is separate from Quest Pulse and the future Compass Book.

## My Quest

My Quest is the active commitment and progress area where players manage current goals, habits, check-ins, contracts, and milestones.

## Quest Leaps

Quest Leaps are a future short-experiment concept for testing ideas and gathering evidence before or during a commitment. Quest Leaps are not implemented in the Quest Pulse rename PR.

## Legacy internal code names

Some internal names remain unchanged for compatibility and to avoid broad mechanical renames in a small product-language PR, including `QuestCompassModal`, `openQuestCompassFromMobileMenu`, the `quest-compass` module path, CSS classes, and the My Quest submenu action id `quest-compass`. These names refer to the existing Quest Pulse implementation and must not be used as a namespace for future Compass Book code.
