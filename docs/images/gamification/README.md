## Contributing Screenshots

To add screenshots:

1. Capture screenshots at 1200x800px minimum resolution
2. Use high-quality PNG format with transparency where appropriate
3. Name files according to the convention above
4. Include both light and dark theme versions if applicable
5. Add descriptive alt text in documentation
